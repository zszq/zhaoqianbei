# zhaoqianbei完整移动端前端项目

完整的已上线移动端前端项目（已获取上传许可）。基于Vue2.0的移动端项目 ，注释详细方便后面学习的人，欢迎star。

## 技术栈

- vue-cli
- vue
- vue-router
- vuex
- axios
- better-scroll
- mint-ui框架的Picker组件和Datetime picker组件
- stylus
- webpack
